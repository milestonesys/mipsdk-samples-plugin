# Demo Access Control Server Application

The Demo Access Control Server application simulates a real-world access control system. It is designed to be 
used with the DemoAccessControlPlugin.

![Demo App](ac_demoapp.png)

For further details about how to use this simulator, please refer to <a href="https://github.com/milestonesys/mipsdk-samples-plugin/tree/main/DemoAccessControlPlugin" target="_blank">DemoAccessControlPlugin in Github</a> or <a href="https://doc.developer.milestonesys.com/mipsdk/index.html?base=samples/pluginsamples/demoaccesscontrolplugin/readme.html&tree=tree_1.html" target="_blank">DemoAccessControlPlugin on MIP SDK Documentation portal</a>.
